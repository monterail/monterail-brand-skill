Communication

Brand Voice

Use this language

  • Partnership-oriented — "together", "our collaboration"
  • Value-driven — results, impact, outcomes
  • Direct, one-sentence CTAs
  • Storytelling through case studies
  • Confident, clear statements
  • Expert knowledge shared generously

Avoid this language

  • Transactional — "vendor", "provider", "supplier"
  • Corporate clichés — "synergy", "leverage"
  • Overly salesy copy
  • Passive voice in CTAs
  • Generic tech buzzwords without substance
  • Writing "monterail" (lowercase) in prose

Quality control

Anti-Patterns

Common mistakes the skill prevents — enforced in every generated artifact.

🚫

Wrong red

Never use #ED0133 — it's a deprecated earlier proposal. Always use the official #F50031 (Pantone 485C).

🚫

Wrong font

Never use Poppins or Inter as primary font. The brand content typeface is Basier Circle — loaded via Fontshare CDN.

🚫

Improvised logo

Never approximate the sygnet in CSS or custom SVG. Always use the official pixel-perfect SVG from assets/.

🚫

Capitalization

Never write monterail in lowercase in prose. The lowercase form exists only inside the official logo wordmark graphic.

🚫

Gradients & pill buttons

Flat, solid colors only. Buttons use border-radius: 8px — never border-radius: 999px pill shapes.

🚫

Wrong grey

Grey Light is #F3F5F6, not #EDF0F2. Use exact values from the official Monterail color palette.
